# Modern Bootstrap Login/Register System

This project contains a responsive login and multi-step registration system built with Bootstrap 5, featuring a modern design with enhanced UI/UX.

## Project Structure

```
.
├── index.html              # Login page
├── register1.html          # Registration step 1: Personal Information
├── register2.html          # Registration step 2: Account Security
├── register3.html          # Registration step 3: Preferences
├── register4.html          # Registration step 4: Confirmation
├── css/
│   └── style.css           # Custom styling
├── js/
│   └── main.js             # Custom JavaScript
└── assets/
    └── brand-logo.png      # Brand logo placeholder
```

## Features

- **Modern UI Design**: Clean, contemporary interface with gradient backgrounds and smooth animations
- **Fully Responsive**: Mobile-first approach with appropriate breakpoints for all device sizes
- **Multi-step Registration**: Progressive form collection with visual progress indicator
- **Enhanced UX**: Improved form elements, better visual hierarchy, and intuitive navigation
- **Consistent Styling**: Unified color scheme, typography, and design language across all pages
- **Accessibility**: Proper semantic HTML, focus states, and form labeling
- **Performance Optimized**: Lightweight implementation with minimal dependencies

## Pages

1. **index.html** - Modern login page with email/password fields
2. **register1.html** - Personal information collection
3. **register2.html** - Account security setup (password and security questions)
4. **register3.html** - Preference selection
5. **register4.html** - Review and confirmation

## Design Elements

- **Color Scheme**: Modern purple/blue gradient (#4361ee to #3a0ca3)
- **Typography**: Poppins font family for a clean, modern look
- **Visual Elements**: 
  - Animated progress indicators
  - Custom form elements with enhanced styling
  - SVG icons for visual interest
  - Gradient backgrounds on left panels
  - Subtle hover and focus effects
- **Layout**: 
  - Split-screen design on larger screens
  - Single-column layout on mobile devices
  - Consistent spacing and alignment

## Customization

To customize this project:

1. Replace `assets/brand-logo.png` with your actual logo
2. Modify colors in `css/style.css` by changing the CSS variables
3. Adjust form fields as needed for your specific requirements
4. Add backend integration to handle form submissions

## Browser Support

This project works on all modern browsers that support Bootstrap 5:
- Chrome
- Firefox
- Safari
- Edge

## Dependencies

- Bootstrap 5 (loaded via CDN)
- Google Fonts (Poppins)
- No additional JavaScript libraries required
